### Vim notes

* `Ctrl + P` - Open file search window
* `Ctrl + P => Ctrl + v` - Open selected file in vertical split
* `Ctrl + P => Ctrl + x` - Open selected file in horizontal split
* `ci'` - Change inside ''
* `ci"` - Change inside ""
* `yi"` - Yank inside "" (コピー)
* `ci<` - Change inside <>
* `Ctrl + n` - Search css property name
* `cc` - Change entire line
