### Tmux Notes

* `tat`(プロジェクトのルートフォルダ) - Open tmux for current folder
* `Ctrl B + %` - Vertical split pane
* `Ctrl B + "` - Horizontal split pane
* `Ctrl B + x` - Delete pane
* `Ctrl B + n` - Go to next window
* `Ctrl B + p` - Go to previous window
* `Ctrl B + z` - Zoom
* `Ctrl B + c` - New window
